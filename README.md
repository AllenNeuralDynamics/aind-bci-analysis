### Install

To pip install:
```
pip install git+https://github.com/kpdaie/BCI_analysis.git
```

To develop the code base:

- clone from github 

```
git clone https://github.com/kpdaie/BCI_analysis.git
```

- and pip install the development environment:

```
pip install -e .[dev]
```
